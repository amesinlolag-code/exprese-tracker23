# Ambient background video (optional)

Drop an .mp4 file named exactly `ambient-bg.mp4` in this folder (`frontend/public/`) to have it
autoplay, muted and looped, behind the login/register screens.

Good free, legal sources for a subtle looping ambient clip:
- https://www.pexels.com/videos/ (search "abstract particles", "night sky", "space")
- https://pixabay.com/videos/

Pick something short (5-15s loops well), dark/low-contrast so text stays readable, and check the
license terms on the download page (Pexels/Pixabay's own licenses are free for this kind of use).

If you don't add a video, the app falls back automatically to a generated animated gradient —
nothing breaks either way.
